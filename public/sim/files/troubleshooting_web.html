Quick Checks

Start here

First Aid

  • Reboot device and networking gear (modem/router).
  • Check physical connections / airplane mode.
  • Try another account/device to isolate scope.
  • Test on a different network (hotspot).
  • Note exact error text and timestamp.

Service Status

Check SaaS status pages and internal outage channels. Capture screenshots for ticketing.
What to collect
  • URL / system, time window, impact (users/teams), repro steps.
  • Client versions (browser, OS, app build).
  • Recent changes (updates, new software, policy).
